PhotoObj

The list below shows the columns in the photoObj view that are most commonly searched. For a complete list, see the Schema Browser .

objID The SDSS ID number for the object
run The Run number (used to find an image in the Get Images tool)
rerun The Rerun Number (when the SDSS software processed the object's data)
camcol The Camera column (used to find an image in the Get Images tool)
field Field Number (used to find an image in the Get Images tool)
type Object type (0 = unknown, 3 = galaxy, 5 = previously known object, 6 = star)
modelmag_u model (computer predicted) magnitude in u filter
modelmag_g model (computer predicted) magnitude in g filter
modelmag_r model (computer predicted) magnitude in r filter
modelmag_i model (computer predicted) magnitude in i filter
modelmag_z model (computer predicted) magnitude in z filter
ra right ascension (sky longitude)
dec declination (sky latitude)
u, g, r, i, z final magnitudes (model magnitudes corrected for reddening)
